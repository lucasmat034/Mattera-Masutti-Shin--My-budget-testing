# MyBudget - Gestionnaire de Budget Personnel

Application de gestion de budget personnel développée en TDD/BDD.

## Installation

```bash
pip install -e .
```

## Utilisation

```bash
mybudget --help
```

## Tests

```bash
pytest
```
